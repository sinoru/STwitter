  • calibrate(passwordLength:saltLength:algorithm:derivedKeyLength:msec:)

    Determines the (approximate) number of iterations of the key derivation algorithm that need to be run to achieve a particular delay (or calculation time).

  • Parameters:

    • passwordLength: Password length in bytes
    • saltLength: Salt length in bytes
    • algorithm: The PseudoRandomAlgorithm to use
    • derivedKeyLength: The desired key length
    • msec: The desired calculation time

    • Returns

    The number of times the algorithm should be run

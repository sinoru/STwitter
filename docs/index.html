STwitter

STwitter is a High Level Twitter Frameworks for Swift.

Features

  • BSD License
  • High-Level Twitter API

========

Usage

Using Swift Package Manager If you want to use as xcodeproj, using swift package generate-xcodeproj command

Requirements

  • Swift 3.0.1
  • Currently only works on Apple Platform, will be work on Linux too.
